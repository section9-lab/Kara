# Kara

Kara is a native macOS assistant for real-time voice notes.

It helps you turn live conversations, meetings, and spoken ideas into clean, editable notes while keeping the experience lightweight and local-first.

## What Kara Does

- Records speech from your microphone
- Transcribes speech into readable text in real time
- Translates spoken content while you work
- Keeps recent sessions organized
- Lets you edit, review, and reuse captured notes
- Runs as a quiet macOS menu bar companion

## Who It Is For

Kara is designed for people who think, meet, and create through conversation:

- Founders and operators capturing meeting notes
- Builders recording product discussions
- Researchers collecting spoken observations
- Bilingual users who need fast transcription and translation
- Anyone who wants spoken ideas to become usable text

## macOS App

Kara is built as a native macOS app with Swift and SwiftUI.

The latest DMG is published from GitHub Releases whenever changes land on `main`.
